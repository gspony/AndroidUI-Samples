## 第一种  TextView 效果库的使用

### 使用库1

#### 库名

```groovy
com.android.util:xselector:1.0.2
```

#### 使用方式

1. 添加依赖

   ```groovy
   implementation 'com.android.util:xselector:1.0.2'
   ```

2. 实例化 XSelector

   ```java
   XSelector.init(this);
   ```
