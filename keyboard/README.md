## 第一种键盘库的使用

### 使用库1

#### 库名

com.liujc.util:jcdialog:1.0.1

#### 使用方式

1. 添加依赖

   ```groovy
   implementation 'com.liujc.util:jcdialog:1.0.1'
   ```

2. 实例化 DialogUtils

   ```java
   DialogUtils.init(this);
   ```
